# PetRescueSystem
本系统是一套面向流浪动物救助站、志愿者及爱心人士的数字化管理平台，致力于提升救助效率与领养透明度。系统涵盖动物档案、救助报告、医疗跟踪、领养全流程审核、志愿者任务调度、社区互动及捐赠物资管理七大模块。普通用户可浏览待领养动物、提交救助线索并参与社区讨论；志愿者可认领任务并记录服务时长；站长拥有最高管理权限，统筹运营与数据看板。系统采用 Spring Boot + Vue 前后端分离架构，支持移动端与后台双端访问，实现救助流程可追溯、物资去向可查询、领养回访有保障。通过信息化手段，平台有效降低救助站管理成本，凝聚社会爱心力量，推动流浪动物救助事业规范化发展。
